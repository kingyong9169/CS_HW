표 관련 태그 실습

HTML 태그
종류 태그 설명
제목 태그 h1~h6 제목을 나타내는 태그, 숫자가 작을수록 큰 제목
본문 관련 태그 p 단락paragraph을 나타내는 태그
br 줄바꿈 태그
hr 수평선을 긋는 태그
목록 관련 태그 ul 순서가 없는 목록을 나타내는 태그
ol 순서가 있는 목록을 나타내는 태그
li ul,ol태그 내부에서 사용되며 목록들의 아이템을 나타내는 태그
표 관련 태그 table 표를 나타내는 태그, 내부에 thread, tbody,tfoot,tr과 같은 태그가 사용
caption 표의 캡션caption을 나타내는 태그
thead 표의 헤더를 나타내는 태그
tbody 표의 본문을 나타내는 태그
tfoot 표의 푸터footer를 나타내는 태그
tr 행row을 나타내는 태그
th 표의 제목 셀cell 나타내는 태그
td 표의 데이터cell 셀을 나타내는 태그
공간 분할 태그 div block형식의 공간을 생성하는 태그
span inline형식의 공간을 만드는 태그
시맨틱 구조 태그 header 페이지의 헤더 영역
nav 네비게이션 영역
aside 사이드에 위치하는 공간
section 여러 중심 내용(article 등)을 감싸는 공간
article 주요 글이 위치하는 공간
footer 페이지의 푸터 영역
이미지 태그 src 이미지의 주소를 지정
alt 이미지를 로드할 수 없을 경우 나오는 글자 지정
width 이미지의 너비 지정
height 이미지의 높이 지정
오디오 태그 src 음악 파일의 경로 지정
preload 음악을 재생하기 전에 모두 불러올지 지정
autoplay 자동 재생 여부 지정
loop 반복 재생 여부 지정
controls 음악 재생 도구 출력 여부 지정
비디오 태그 src 동영상 파일의 경로 지정
poster 동영상 준비 중일 때의 이미지 파일의 경로 지정
preload 비디오를 재생하기 전에 모두 불러올지 지정
autoplay 자동 재생 여부 지정
loop 반복 재생 여부 지정
controls 음악 재생 도구 출력 여부 지정
width 동영상의 너비 지정
height 동영상의 높이 지정
form 태그 form 입력 양식을 만들기 위한 태그
action 입력 데이터의 전달 위치 지정
method 입력 데이터의 전달 방식 지정
input 태그 type 입력 양식의 종류를 결정하는 속성값
name 서버로 양식을 전송할 때, 각 입력항목을 구분하기 위한 이름값 설정
placeholder 사용자가 값을 입력하지 않았을 때 안내 글을 작성
disabled 입력 양식을 비활성화
readonly 입력 양식에 값을 쓸 수 없고 읽기만 가능하게 함
value 입력 양식의 값
textarea 태그 cols 태그의 너비를 지정(하나의 라인에 입력되는 글자 수)
rows 태그의 높이를 지정(라인 수)
name input태그의 name속성과 동일
placeholder input태그의 placeholder속성과 동일
disabled input태그의 disabled속성과 동일
select 태그 select 선택 양식을 생성
optgroup 옵션을 그룹화
option 선택할 수 있는 옵션을 생성
name input태그의 name속성과 동일
multiple 여러 옵션을 선택할 수 있게 함
disabled input태그의 disabled속성과 동일
label 태그 for label과 연결할 입력 양식 태그를 설정(입력 양식의 id속성값)
fieldset 태그 fieldset 입력 양식들을 하나의 그룹으로 묶기 위해 사용
legend 태그 legend fieldset에 대한 설명을 제공
